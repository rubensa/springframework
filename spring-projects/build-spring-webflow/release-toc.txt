SPRING WEB FLOW (SWF) - 1.0 Release archive table of contents

Directory | Description
-----------------------
. | contains project distribution units (jars and source zip archives)
docs | contains the reference manual and API Javadocs
projects | contains buildable Spring Web Flow projects, including sample applications
projects/build-spring-webflow | contains the master build file to build all projects
projects/common-build | contains the ant 1.6 "common build system" used by all projects to compile/build/test
projects/repository | contains Spring Web Flow project dependencies (jars)
projects/spring-binding | contains buildable Spring Binding project sources, an internal library used by SWF
projects/spring-webflow | contains buildable Spring Web Flow project sources
projects/spring-webflow-samples | contains buildable Spring Web Flow sample application sources

Each source project like "spring-webflow" and "spring-webflow-samples/phonebook" is directly
importable as an Eclipse project.

See the readme.txt within the above directories for additional information.
