Contains interfaces and classes for registering and refreshing web flows in a registry.

You can construct a generic, initially empty FlowRegistry, and then populate it with Flow definitions using a FlowRegistrar as follows: 

    BeanFactory beanFactory = ...
    FlowRegistryImpl registry = new FlowRegistryImpl();
    Resource[] locations = new Resource[] {
       new FileSystemResource("WEB-INF/flowA.xml"),
       new FileSystemResource("WEB-INF/flowB.xml")
    };
    XmlFlowRegistrar registrar = new XmlFlowRegistrar(new DefaultFlowArtifactFactory(beanFactory), locations);
    registrar.registerFlowDefinitions(registry);
